How It Works

The habit, not the hack.

1

Copy the leak

Your developer copies a JSON error log, a CSV export, or a terminal stacktrace. It contains real Aadhaar numbers, PANs, emails, or API keys.

2

fz clip intercepts

Fortress Zero detects the format (JSON, CSV, Prose, Code, Log), validates entities with Verhoeff and Luhn checks, applies context guards (never touching variable names or localhost), and generates a deterministic synthetic twin.

3

Paste with proof

Clean data hits the AI tool. A tamper-evident attestation log is written locally. When the auditor asks, you prove raw PII never left the endpoint.
